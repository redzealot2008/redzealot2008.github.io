pwconv命令详解

打开用户的影子密码

语法

pwconv

功能介绍

Linux 系统里的用户和组密码，分别存在 /etc目录下的 passwd 和 group 文件中。因系统刚运作所需，任何程序都可读取它们，这造成安全上的漏洞。影子密码将文件内的密码改存在 /etc 目录下的 shadown 和 gshadow 文件内，只允许系统管理者读取，同时把原密码设置为“x”字符，有效地强化了系统的安全性。影子密码的功能可随时打开或关闭，只需要执行 pwconv 命令就能打开组影子密码。

参数

无

例子

打开用户的影子密码，加强安全防护：

# pwconv
