dip命令详解

IP拨号连接

语法

dip [-aikltv][-m][-p<协议>][拨号script文件]

功能介绍

dip可控制调制解调器，以拨号IP的方式建立对外的双向连接。

参数

-a 询问用户名称与密码。

-i 启动拨号服务器功能。

-k 删除执行中的dip程序。

-l 指定要删除的连线，必须配合-k参数一起使用。

-m 设置最大传输单位，预设值为296。

-p<协议> 设置通信协议。

-t 进入dip的指令模式。

-v 执行时显示详细的信息。
