dd命令详解

读取、转换并输出数据

语法

dd [bs= <字节数>] [cbs=<字节数>] [conv=<关键字>] [count=<块数>] [ibs=<字节数>] [if=<文件>] [obs=<字节数>] [of=<文件>] [seek=<块数>] [skip=<块数>] [–help] [–version]

功能介绍

dd 可从标准输入或文件读取数据，根据指定的格式来转换数据，在输出到文件、设备或标准输出。

参数

bs 将ibs（输入）或osb（输出）设成指定的字节数

cbs 每次只转换指定的字节数

conv 指定文件转换的方式。

count 仅读取指定的块数

ibs 每次读取的字节数

if 从文件读取。若未指定此参数，则从标准输入设备读取数据

obs 每次输出的字节数

of 输出至文件。若未指定此参数，则输出至屏幕

seek 一开始输出时，跳过指定的块数。块与obs的大小相同

skip 一开始读取时，跳过制定的块数。块与ibs的大小相同
