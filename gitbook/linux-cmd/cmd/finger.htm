finger命令详解

查找并显示用户信息

语法

finger [-lmps] [账号名称…]

功能介绍

finger 命令会去寻找并显示指定账号的用户相关信息，包括本地与远程主机的用户皆可，账号名称名优大小写的差别。单独执行 finger 命令，它会显示本地主机现在所有用户的登录信息，包括账号名称，真实名称、登录终端，空闲时间、登录时间，以及地址和电话。

参数

参数 作用
-l 列出该用户的账号名称、真实姓名、用户根目录、登录所用的 shell、登录时间、邮件地址、电子邮件状态，还有项目文件（.project）和计划文件（.plan）内容。这项参数为默认值，若仅仅给予 finger命令账号名称，有无加参数效果相同。
-m 配出查找用户的真实姓名。用户通常具有两个名称，一个是登录系统的用户账号，另外一个则是真实姓名。加入没有加上这个参数，finger 命令会去查找所有符合指定账号名称的用户账号与真实姓名，搭配此参数之后，真实姓名将不在查找范围之内
-p 列出该用户的账号名称、真实姓名、用户根目录、登录所用的shell、登录时间、邮件地址、电子邮件状态，但不显示该用户的项目和计划内容
-s 列出该用户的账号名称、真实姓名、登录终端、空闲时间、登录时间，以及地址和电话
